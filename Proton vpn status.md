### Connect
```bash
protonvpn-cli connect 
```
OR
```bash
protonvpn-cli c
```



### Status
```bash
protonvpn-cli status 
```
or
```bash
protonvpn-cli s
```


### Disconnect
```bash
protonvpn-cli disconnect 
```


OR 
```bash
protonvpn-cli d
```


### Killswitch
On
```bash
protonvpn-cli ks --on
```

Permanent
```bash
protonvpn-cli ks --permanent
```

Off

```bash
protonvpn-cli ks --off
```
